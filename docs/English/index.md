```title="被单词时机"
早晨:
	- 早晨人脑意识迷糊, 但干扰大脑思考记忆的factor未"苏醒"(干扰因素还未启动), 因而效率会更高
	- 早上的活动没有"事前干扰", 大脑尚处于非活跃状态
晚上:
	- 晚上的活动没有"事后干扰"
	- 晚上发生高强度刺激的事情的概率较低, 这时候不管是知识的记忆，还是提醒自己事项的记忆都是很牢固的
	- 入眠之前做某些事情一般能被记住, 因为脑细胞会在睡眠期间集中存储记忆信息
		-  如: 睡前提醒自己第二天要带某本书或者记得跟老总汇报个情况，一般来讲第二天刚醒来都还记得
```
